## Project 1: Salary Dashboard (2023‚ÄìPresent)

This project features an interactive Excel dashboard designed to analyze starting salaries for data-related jobs. The dashboard allows users to explore how compensation varies by job title, country, and work schedule type, helping job seekers better understand current market trends.

### Key Features
- Interactive dashboard with charts and filters
- Salary analysis by job title, country, and schedule type
- Median salary calculations
- Data validation for controlled user input

### Tools & Skills Used
- Excel Charts (Bar Charts, Map Charts)
- Formulas & Functions (`MEDIAN`, `IF`, `SEARCH`, `FILTER`)
- Data Validation
- Power Query (for data preparation)

### Methodology
1. Imported and cleaned job salary data in Excel.
2. Created helper tables and formulas to calculate median salaries.
3. Built visualizations to compare salaries across roles and locations.
4. Applied data validation to improve dashboard usability.
5. Designed a clean dashboard layout for easy interpretation.

## Project 2: Data Science Job Market & Skills Analysis

### Download
[üì• Download Project 2 Excel File](Project2_SkillsAnalysis.xlsx)



As a current job seeker, I wanted to better understand which skills and roles lead to higher pay in the data science job market. This project analyzes job postings to uncover relationships between skills, salaries, and geographic location using advanced Excel modeling techniques.

### Questions Explored
1. Do roles requiring more skills offer higher pay?
2. How do salaries differ across regions?
3. What are the most in-demand skills for data professionals?
4. Which skills are associated with the highest salaries?

### Tools & Skills Used
- Power Query (ETL)
- PivotTables & PivotCharts
- Power Pivot (Data Modeling)
- DAX (Measures & Calculations)
- Advanced Excel Charting

---

### 1Ô∏è‚É£ Do More Skills Lead to Higher Pay?

**Approach**
- Used Power Query to extract and clean job and skill data.
- Created separate tables for job details and job skills.
- Prepared the data for modeling and analysis.

**Insights**
- Roles requiring a larger number of skills tend to have higher median salaries.
- Senior-level technical roles show the strongest correlation between skill count and pay.
- Less specialized roles generally offer lower compensation.

**Approach**
